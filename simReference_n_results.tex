\chapter{Simulation Results}
\label{chap:simulation_results}

\section{Reference Geometry and Physical Consistency Checks}
\label{sec:results_validation_strategy}

The simulations are used to test whether the numerical model reproduces the expected behaviour of a retrievable collective spin-wave mode. The central observable is the normalized fibre-coupled retrieval,
\begin{equation}
    \eta_{\mathrm{fib}}(t)
    =
    \frac{P_{\mathrm{fib}}(t)}
    {P_{\mathrm{tot}}(0)},
    \label{eq:results_normalized_retrieval}
\end{equation}
where \(P_{\mathrm{fib}}(t)\) is the power coupled into the target fibre mode after storage time \(t\), and \(P_{\mathrm{tot}}(0)\) is the total emitted power at zero storage time. This normalization removes the arbitrary absolute scale of the emitted field and isolates the decay of the mode that remains retrievable into the selected optical channel.

The quantity \(\eta_{\mathrm{fib}}(t)\) is not an absolute memory efficiency. It does not include the full storage process, optical-depth dependence or  propagation through the medium. It instead is interpreted as a normalized mode-matched retrieval proxy where it quantifies how much of the initially retrievable collective mode remains coupled to the selected fibre mode after a storage time \(t\).

The reference geometry is based on the standalone \({}^{133}\mathrm{Cs}\) warm-vapour memory reported by Jutisz \emph{et al.}~\cite{Smqms_JutiszEtAl_2025}. The simulated cell is a cylindrical vapour cell with length \(\SI{75}{\milli\metre}\) and diameter \(\SI{4}{\milli\metre}\), operated near \(\SI{75}{\celsius}\) with nitrogen buffer gas. The signal and control fields are represented as Gaussian beams with FWHM diameters of \(\SI{300}{\micro\metre}\) and \(\SI{120}{\micro\metre}\), respectively. Since the optical mode occupies only a small fraction of the full transverse cell area, the relevant simulated atoms are those with appreciable overlap with the signal and control modes.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/ensemble_schematic.jpeg}
    \caption{Reference warm-vapour geometry. The signal and control beams define a localized optical interaction region inside the cylindrical vapour cell.}
    \label{fig:ensemble_schematic}
\end{figure}

This geometry provides a useful reference case because the expected physical trends are well established. Buffer gas is expected to slow motional escape from the optical mode, larger beams should extend the overlap time, magnetic-field gradients should dephase the ground-state coherence, and phase mismatch should reduce directional retrieval. The simulations are therefore used as physical consistency checks of the implemented retrieval-overlap model, rather than as a quantitative validation of the complete experimental memory.

\section{Directional Collective Emission}
\label{sec:results_directional_collective_emission}

During retrieval, the read control field converts the stored spin wave into an optical polarization. Each atom contributes a field amplitude, and the useful retrieved field is obtained from the coherent sum of these amplitudes. Following the spin-wave phase-matching condition~\cite{PsiL-odamIFm_GorshkovEtAl_2007}, the retrieved optical wave vector is
\begin{equation}
    \mathbf{k}_{\mathrm{out}}
    =
    \mathbf{k}_{\mathrm{sig}}^{\mathrm{write}}
    -
    \mathbf{k}_{\mathrm{c}}^{\mathrm{write}}
    +
    \mathbf{k}_{\mathrm{c}}^{\mathrm{read}},
    \label{eq:results_phase_matching}
\end{equation}
where \(\mathbf{k}_{\mathrm{out}}\) is the retrieved optical wave vector, \(\mathbf{k}_{\mathrm{sig}}^{\mathrm{write}}\) is the signal wave vector during storage, and \(\mathbf{k}_{\mathrm{c}}^{\mathrm{write}}\) and \(\mathbf{k}_{\mathrm{c}}^{\mathrm{read}}\) are the write and read control-field wave vectors.

In the simulation, the far-field amplitude in a trial emission direction \(\mathbf{k}\) is represented by a coherent weighted dipole sum,
\begin{equation}
    E(\mathbf{k},t)
    \propto
    \sum_{j}
    w_j(t)
    \exp\!\left[
        i\Phi_j(t)
        -
        i\mathbf{k}\cdot\mathbf{r}_j(t)
    \right],
    \label{eq:results_coherent_dipole_sum}
\end{equation}
where \(\mathbf{r}_j(t)\) is the position of atom \(j\), \(w_j(t)\) contains the spatial mode weight and possible wall-survival factor, and \(\Phi_j(t)\) contains the stored spin-wave phase together with any additional phase accumulated during storage. The directional structure of the emitted field is therefore determined by whether the phase factors in Eq.~\eqref{eq:results_coherent_dipole_sum} add constructively in the selected optical mode.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.70\textwidth]{Figures/far_field_angular_emission_profile.jpeg}
    \caption{Far-field angular emission profile obtained from the coherent dipole sum. The strongest emission is concentrated around the forward phase-matched direction.}
    \label{fig:far_field_angular_emission_profile}
\end{figure}

The simulated far-field distribution in Fig.~\ref{fig:far_field_angular_emission_profile} is peaked around the forward phase-matched direction. This angular concentration supports the interpretation that the retrieved signal is governed by collective interference. Away from the central lobe, residual emission arises from imperfect cancellation into other angular modes, while the dominant contribution remains aligned with Eq.~\eqref{eq:results_phase_matching}. If the spin-wave phases are randomized, the atoms may still emit, but the fields no longer add coherently into the selected mode. Directional emission is therefore a useful diagnostic of the phase structure carried by the stored spin wave.

\section{Monte Carlo Convergence}
\label{sec:results_monte_carlo_convergence}

The retrieval amplitude is estimated from a finite number of sampled atoms and stochastic trajectories. Numerical convergence must therefore be checked before physical trends are interpreted. Independent Monte Carlo realizations are expected to fluctuate around a stable ensemble mean, while the global decay should remain unchanged once a sufficient number of samples has been used.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/monte_carlo_convergence.jpeg}
    \caption{Monte Carlo convergence of the normalized retrieved signal. Individual stochastic realizations fluctuate around the same global decay trend as the ensemble mean.}
    \label{fig:monte_carlo_convergence}
\end{figure}

In Fig.~\ref{fig:monte_carlo_convergence}, the individual realizations follow the same decay envelope, with fluctuations caused by different random trajectories. The ensemble mean is stable, indicating that the observed decay is not dominated by rare trajectories.. Increasing the number of simulated atoms therefore improves the smoothness of the curve without changing the converged physical trend.

\section{Gas-Phase Transport and Buffer-Gas Pressure}
\label{sec:results_buffer_gas_pressure}

Buffer gas modifies the motion of alkali atoms through velocity-changing collisions. In a warm vapour, atoms carrying the stored coherence can leave the high-overlap region defined by the signal and control beams. Once this happens, their contribution to \(P_{\mathrm{fib}}(t)\) is reduced, even if the internal ground-state coherence has not been destroyed.

The diffusion model represents this transport through the diffusion coefficient \(D\). Higher buffer-gas pressure reduces \(D\), slows spatial escape from the optical mode, and increases the time over which atoms remain mode matched to the readout field. A slower decay of \(\eta(t)\) is therefore expected when the pressure is increased.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/pressure_scan_coupling_decay.jpeg}
    \caption{Pressure scan of the normalized fibre-coupled retrieval. Increasing the buffer-gas pressure reduces diffusion and extends the time over which atoms remain inside the high-overlap optical mode.}
    \label{fig:pressure_scan_coupling_decay}
\end{figure}

The pressure scan in Fig.~\ref{fig:pressure_scan_coupling_decay} follows this expected trend. At lower pressure, the retrieval decreases more rapidly because atoms diffuse away from the region that overlaps efficiently with the signal, control, and fibre modes. At higher pressure, the motion is slowed and the retrievable collective mode survives for a longer time. The decay should be interpreted as a loss of mode overlap rather than as the immediate destruction of the atoms. Atoms outside the beam can still carry local coherence, but they no longer contribute efficiently to the phase-matched fibre mode.

\section{Beam-Size Dependence}
\label{sec:results_beam_size}

The spatial-overlap mechanism can be isolated by changing the optical beam sizes. A wider beam defines a larger interaction region, so a diffusing atom must travel farther before its coupling to the stored and retrieved modes becomes small. For fixed optical preparation and transport parameters, larger beams should therefore increase the decay time of the retrieved mode.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/beam_size_scan_coupling_decay.jpeg}
    \caption{Beam-size dependence of the normalized retrieval. Larger optical beams increase the spatial region over which atoms remain coupled to the readout mode, producing a slower decay of \(P_{\mathrm{fib}}(t)/P_{\mathrm{tot}}(0)\).}
    \label{fig:beam_size_scan}
\end{figure}

The behaviour in Fig.~\ref{fig:beam_size_scan} is consistent with transit-time intuition in warm-vapour EIT systems~\cite{Apgteitiav_FinkelsteinEtAl_2023}. Enlarging the optical mode reduces the sensitivity of the stored spin wave to short diffusive displacements, and the fibre-coupled retrieval decays more slowly. The result reflects the survival of spatial mode overlap. In an experiment, increasing the beam size may also reduce optical intensity for fixed power, modify the EIT bandwidth, and change the effective optical depth sampled by the mode. These additional optical effects are not included in the present retrieval-only model.

\section{Comparison with a Cold Gaussian Ensemble}
\label{sec:results_bec_comparison}

The same retrieval-overlap formalism was applied to a cold Gaussian ensemble. The simulated cloud contains \(10^8\) atoms with a Gaussian spatial distribution of width approximately \(\SI{1}{\milli\metre}\). The signal and control beams are smaller than the cloud, with dimensions of waist of  $\SI{500}{\micro\meter}$ and $\SI{150}{\micro\meter}$ , so the stored mode is still selected by the optical overlap rather than by the full atomic density distribution.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/bec_beam_size_scan.jpeg}
    \caption{Beam-size scan for a cold Gaussian ensemble. Increasing the optical mode size increases the coupling time, although the effect is weaker than in the warm-vapour reference case.}
    \label{fig:bec_beam_size_scan}
\end{figure}

In Fig.~\ref{fig:bec_beam_size_scan}, larger beams also increase the coupling time in the cold ensemble. The dependence is weaker than in the warm-vapour reference geometry because rapid thermal diffusion is no longer the dominant limitation. In the warm vapour, transverse beam size directly controls how quickly atoms leave the optical mode. In the cold cloud, the atoms move more slowly, and the decay is less strongly governed by diffusive transit through the beam. The same coherent-overlap model can therefore describe different platforms, while the dominant loss mechanism depends on the motional regime.

\section{Geometric Phase Mismatch from Control-Beam Angle}
\label{sec:results_control_beam_angle}

A relative angle between the signal and control fields changes the spin-wave wave vector. For the write process, the wave-vector mismatch is~\cite{PsiL-odamIFm_GorshkovEtAl_2007}
\begin{equation}
    \Delta\mathbf{k}
    =
    \mathbf{k}_{\mathrm{sig}}^{\mathrm{write}}
    -
    \mathbf{k}_{\mathrm{c}}^{\mathrm{write}},
    \label{eq:results_delta_k_angle}
\end{equation}
where \(\mathbf{k}_{\mathrm{sig}}^{\mathrm{write}}\) and \(\mathbf{k}_{\mathrm{c}}^{\mathrm{write}}\) are the signal and control wave vectors during storage. A nonzero angle introduces a transverse spin-wave phase grating. Atoms moving through this grating acquire position-dependent phases. The shorter the spin-wave period, the smaller the displacement required to produce a significant phase change.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/control_beam_angle_scan.jpeg}
    \caption{Control-beam angle scan. Changing the signal-control angle modifies the spin-wave wave vector and changes the sensitivity of the stored coherence to atomic motion.}
    \label{fig:control_beam_angle_scan}
\end{figure}

The angle scan in Fig.~\ref{fig:control_beam_angle_scan} indicates faster decay when the spin-wave phase varies more rapidly across the ensemble. The mechanism is loss of relative phase rather than loss of atomic coherence. Atoms may still carry ground-state coherence, but the spatial phase pattern no longer overlaps efficiently with the phase-matched readout mode. This behaviour is consistent with the use of nearly co-propagating signal and control fields in warm-vapour EIT, where a small two-photon wave vector reduces motional dephasing~\cite{Apgteitiav_FinkelsteinEtAl_2023}.

\section{Magnetic-Gradient Dephasing}
\label{sec:results_magnetic_gradient}

Magnetic-field gradients dephase the stored coherence by making the ground-state splitting position dependent. The spatially uniform part of the splitting produces only a common phase and does not reduce retrieval. The relevant quantity is the differential shift \(\Delta\omega_{sg}(\mathbf{r})\), which gives the accumulated phase
\begin{equation}
    \phi_j(t)
    =
    \int_0^t
    \Delta\omega_{sg}\!\left(\mathbf{r}_j(t')\right)
    \,dt',
    \label{eq:results_magnetic_phase}
\end{equation}
where \(t'\) is the integration variable and \(\mathbf{r}_j(t')\) is the trajectory of atom \(j\). If \(\Delta\omega_{sg}(\mathbf{r})\) is constant across the ensemble, all atoms acquire the same additional phase. If it varies spatially, different atoms accumulate different phases and the collective readout is reduced.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/magnetic_gradient_scan.jpeg}
    \caption{Magnetic-gradient-induced decay of the normalized retrieval. A spatially varying ground-state splitting causes different parts of the ensemble to accumulate different phases, reducing constructive retrieval into the target mode.}
    \label{fig:magnetic_gradient_scan}
\end{figure}

The magnetic-gradient scan in Fig.~\ref{fig:magnetic_gradient_scan} gives a faster decay as the gradient is increased. This loss channel is distinct from beam-overlap decay. Atoms can remain inside the optical mode and still stop contributing constructively if their internal phases become spatially scrambled. Spatial escape can be mitigated by buffer gas or larger beams, whereas magnetic dephasing requires field homogeneity and magnetic shielding.

\section{Compact-Cell Geometry}
\label{sec:results_compact_cell_geometry}

The reference cell is much larger than the optical mode in the transverse direction. In that regime, atoms can diffuse out of the beam before wall collisions dominate the retrieval decay. A compact cell changes this balance because the optical mode occupies a substantial fraction of the cell cross-section. The simulated compact geometry has diameter \(\SI{1}{\milli\metre}\) and length \(\SI{10}{\milli\metre}\), with control and signal FWHM diameters of approximately \(\SI{0.8}{\milli\metre}\) and \(\SI{0.3}{\milli\metre}\), respectively.

The compact cell is not simply a scaled-down version of the reference vapour cell. Because the optical mode occupies a substantial fraction of the transverse cross-section, atoms that leave the high-overlap region encounter the wall on comparable timescales. The dominant question therefore changes from how long atoms remain inside the beam to whether they remain coherent after boundary encounters.

In the compact geometry, the wall becomes part of the storage dynamics. An atom carrying spin-wave coherence can reach the boundary on the same timescale over which the retrieval signal decays. The boundary condition must therefore specify both the mechanical trajectory after a wall encounter and the survival of the internal coherence. This distinction becomes central when anti-relaxation coatings are represented as effective wall-survival parameters.

\section{Wall Coatings as an Effective Boundary Condition}
\label{sec:results_wall_coatings}

A wall encounter affects the atom in two different ways. Mechanical reflection determines whether the atom remains inside the vapour volume. Internal-state relaxation determines whether the atom still carries the ground-state coherence required for spin-wave retrieval. These processes are physically distinct. An atom can return to the vapour after a wall collision while its spin coherence has already been randomized by the surface interaction~\cite{OP_Happer_1972}.

The coating is represented by an effective coherence-survival probability per wall collision. If \(N_{\mathrm{b}}\) is the mean number of wall collisions before depolarization, the survival probability for a single collision is
\begin{equation}
    p_{\mathrm{survive}}
    =
    1-\frac{1}{N_{\mathrm{b}}},
    \label{eq:results_wall_survival_probability}
\end{equation}
where \(p_{\mathrm{survive}}\) is the probability that the atomic ground-state coherence survives one wall encounter. This phenomenological model does not describe the microscopic atom--surface chemistry of a particular coating. It condenses the surface interaction into a single effective parameter that controls how rapidly wall encounters remove atoms from the coherent spin-wave sum.

Anti-relaxation coatings reduce the probability that a wall collision destroys the atomic spin state. Their performance depends on material, temperature, surface coverage, adsorption dynamics, and morphology~\cite{Aiacoavc_ChiEtAl_2020}. In the compact-cell simulations, these microscopic details are represented by scanning \(N_{\mathrm{b}}\). Small \(N_{\mathrm{b}}\) corresponds to a strongly depolarizing wall, while large \(N_{\mathrm{b}}\) corresponds to coherence preservation over many wall collisions.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/coating_bounce_scan.jpeg}
    \caption{Compact-cell coating scan. Increasing the mean number of coherence-preserving wall collisions \(N_{\mathrm{b}}\) improves the retrieval decay when wall relaxation is the dominant loss channel.}
    \label{fig:coating_bounce_scan}
\end{figure}

The compact-cell coating scan in Fig.~\ref{fig:coating_bounce_scan} indicates that increasing \(N_{\mathrm{b}}\) improves the retrieval lifetime when the wall is strongly depolarizing. In this regime, each wall collision has a significant probability of removing the atom from the coherent spin wave, so the wall behaves almost like an absorbing boundary for the retrievable mode. Larger \(N_{\mathrm{b}}\) allows atoms to return to the vapour while retaining coherence, and the fibre-coupled signal decays more slowly.

The improvement saturates once wall-induced depolarization is no longer the dominant limitation. At large \(N_{\mathrm{b}}\), the curves begin to converge because the remaining decay is set mainly by diffusion through the optical mode, finite cell geometry, and imperfect overlap with the readout mode. Beyond that regime, improving the coating alone produces only a limited increase in \(P_{\mathrm{fib}}(t)/P_{\mathrm{tot}}(0)\).

\section{Temperature Dependence in the Compact Cell}
\label{sec:results_temperature_compact_cell}

Temperature changes several physical quantities at once in a warm-vapour memory. Higher temperature increases the alkali vapour pressure and can increase optical depth, which is favourable for storage and retrieval efficiency. It also increases thermal motion, modifies transport in the buffer gas, and can make wall interactions and coating stability more relevant~\cite{Apgteitiav_FinkelsteinEtAl_2023,Aiacoavc_ChiEtAl_2020}.

The temperature scan considered here isolates the transport and wall-relaxation contribution to the retrievable spin-wave mode. The full temperature dependence of the EIT optical depth is not computed. The curves therefore should not be interpreted as a complete optimization of memory efficiency. They quantify how motion and wall encounters affect the mode that remains available for fibre-coupled retrieval.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/compact_temperature_scan.jpeg}
    \caption{Temperature dependence of the compact-cell retrieval for different wall-survival parameters. The decay reflects transport and wall-relaxation contributions to the retrievable-mode lifetime.}
    \label{fig:compact_temperature_scan}
\end{figure}

The temperature scan in Fig.~\ref{fig:compact_temperature_scan} indicates that the temperature dependence is strongest when wall survival is low. In that regime, the decay is governed by how rapidly atoms reach the boundary and lose coherence. For high wall survival, the curves become less sensitive to \(N_{\mathrm{b}}\). Atoms may survive wall encounters, but retrieval can still decrease because the atoms return with reduced spatial overlap or with a phase pattern that no longer matches the fibre-coupled mode.

\section{Cell Length and Spin-Wave Phase}
\label{sec:results_cell_length_spin_wave}

The compact-cell geometry also changes how much of the spin-wave phase grating is sampled by the ensemble. The stored spin wave contains the phase factor
\begin{equation}
    e^{i\Delta\mathbf{k}\cdot\mathbf{r}},
    \label{eq:results_spin_wave_phase_factor}
\end{equation}
where \(\Delta\mathbf{k}\) is set by the signal-control geometry. If the cell length is short compared with the spin-wave period, the phase varies only weakly across the ensemble. If the cell spans many spin-wave periods, different longitudinal regions contribute with different phases to the coherent readout.

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.72\textwidth]{Figures/cell_length_spin_wave_scan.jpeg}
    \caption{Cell-length scan relative to the spin-wave phase period. When the cell spans many spin-wave periods, different regions of the ensemble contribute with different phases and the retrievable collective mode is reduced.}
    \label{fig:cell_length_spin_wave_scan}
\end{figure}

The cell-length scan in Fig.~\ref{fig:cell_length_spin_wave_scan} indicates that the long-time plateau depends on the relation between cell length and spin-wave phase period. This is not only a volume effect. A longer cell contains more atoms, but it also samples a larger portion of the spin-wave phase grating. When the phase varies significantly over the cell, different regions interfere less constructively in the target mode, and the coherent retrieval can decrease.

This effect matters when compact and elongated geometries are compared. Miniaturization changes the wall-collision rate, but it also changes the sampled spin-wave phase structure and its overlap with the readout mode. The cell dimensions therefore affect retrieval through both boundary interactions and collective phase matching.
